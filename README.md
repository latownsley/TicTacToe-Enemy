# TicTacToe-Enemy

Made for an end of the year C++ Game Making Challenge, Tic Tac Toe is a simple c++ iteration of Tic Tac Toe played against an enemy AI. Players can play against various difficulties of computer enemies. Each difficulty has been implemented with a different algorithm (Minimax, random placements, etc).


## 2025 C++ Game Making Challenge

Made as part of a 2025 C++ game making challenge.
